# Synchronized Scroll Mobile UI

A Flutter mobile app demonstrating synchronized scrolling between a circle list and grid view. When the top container is visible, both views scroll together. After scrolling past the top, each view scrolls independently.

## Features

- Synchronized scrolling between circle list and grid view
- Smooth transition to independent scrolling modes
- Automatic pagination with scroll position preservation
- Circle selection with grid content refresh
